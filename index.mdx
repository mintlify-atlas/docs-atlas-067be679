  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Download Philo and start journaling with AI-powered widgets in just a few steps.
  </p>

  <Steps>
    <Step title="Download and install Philo">
      Download the latest release for macOS, Windows, or Linux from the [GitHub releases page](https://github.com/ComputelessComputer/philo/releases/latest). Install the application on your system.
    </Step>
    
    <Step title="Add your Anthropic API key">
      Open **Settings** (⌘,) and enter your Anthropic API key to enable AI widget generation. Get your API key from the [Anthropic Console](https://console.anthropic.com/).
    </Step>
    
    <Step title="Start journaling">
      Write your first note in today's entry. Use markdown formatting, add tasks with `- [ ] Task name`, and paste images directly into the editor.
    </Step>
    
    <Step title="Generate your first widget">
      Select text describing what you want (like "calorie tracker" or "habit chart"), then press **⌘↵** to generate an interactive widget powered by Claude.
    </Step>
  </Steps>
</div>
